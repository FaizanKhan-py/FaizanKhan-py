Final Year Project | AI Vehicle Smoke Emission Monitor

> An end-to-end AI system for detecting and monitoring vehicle exhaust smoke emissions using **computer vision** and **deep learning**.
>
> - Detects vehicles in real-time using YOLOv8 object detection
> - Measures exhaust smoke intensity through image analysis
> - Auto-generates evidence-based challans for violations
> - Deployed on cloud for scalability and reliability
>
> `Python` `YOLOv8` `OpenCV` `TensorFlow` `Django` `AWS` `React`
